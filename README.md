## Deskripsi
Proyek ini merupakan aplikasi web sederhana yang terhubung dengan database MySQL
menggunakan Docker Compose dengan arsitektur multi-container.
Aplikasi ini dibuat untuk memenuhi tugas mata kuliah terkait container dan virtualisasi.

## Teknologi yang Digunakan
- Docker
- Docker Compose
- PHP 8.2 (Apache)
- MySQL

## Cara Menjalankan Aplikasi
1. Pastikan Docker Desktop sudah berjalan
2. Buka terminal / PowerShell
3. Masuk ke folder project
cd D:\multi-container-app
4. Jalankan Docker Compose
docker compose up -d --build
5. Buka browser dan akses
http://localhost:8080


## Hasil
Jika berhasil, akan muncul pesan:
